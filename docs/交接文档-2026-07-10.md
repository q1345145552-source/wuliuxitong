# 湘泰物流系统 — 项目交接上下文

> 日期：2026-07-10 | 当前 commit：`6d32bc9`

## 1. 项目目标

湘泰物流网站是中泰跨境物流管理系统，三端合一（管理员/员工/客户端），管理从预报单到签收的完整物流链路。

## 2. 技术栈

| 层 | 技术 |
|---|---|
| 前端 | Next.js 16 + React 19 + TypeScript |
| 后端 | Node.js 自建 HTTP 服务（`apps/api/src/server.ts`）+ tsx 热执行 |
| 数据库 | PostgreSQL（生产 Neon 云库，本地 Docker）|
| ORM | Prisma |
| 部署 | Docker Compose（4 容器：web+api+postgres+redis）|
| AI | DeepSeek API |

## 9. 注意事项

### 高频错误模式
- **60% 的 bug 是同根因**：三端 API 独立编写，改一个漏另外两个
- **改 API 默认值前 grep 全局调用方**（教训 #15）
- **父子运单逻辑**：运单列表看父，装柜看子，尾端派送看子，查重排除自己（教训 #14）
- **分页默认 500 条**，前端三个 service 函数已传 `pageSize=500`

### 三端不可见规则
- 客户端**绝对不能**看到柜号、船次/船名等装柜信息

### 部署
```bash
cd /root/MyWebSite && bash deploy.sh
```
